\section{Architectural Scope}
The follow content describes all the architectural responsibilities which are required to be addressed by all platforms of software being implemented.

\begin{enumerate}
	\item A reliable database which should be secure, reliable (accessible at all times to its users from any location) and efficient on all 		platforms of software being implemented.
	\item Providing a strong reporting infrastructure which should adhere to the following requirements:
	\begin{list}{enumerate}
		\item Consistent
		\item Timely
		\item Cost effective
		\item Accurate
	\end{list}
	\item There should be a clear and definitive means of accomplishing specific tasks which users may implement with ease, assisting them with specific process executions.
\end{enumerate}
