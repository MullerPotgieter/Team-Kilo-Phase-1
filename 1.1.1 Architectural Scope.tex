\section*{Architectural Scope}
The follow content describes all the architectural responsibilities which are required to be addressed by all platforms of software used.

\begin{enumerate}
	\item The database should be efficient on all platforms of software being implemented, secure and reliable.
	\item Providing a strong reporting infrastructure which should adhere to the following requirements:
	\begin{list}{label}{spacing}
		\item Consistent
		\item Timely
		\item Cost effective
		\item Accurate
	\end{list}
	\item There should be a clear and definitive means of accomplishing specific tasks which users can easily adhere to, which would assist them with specific process executions.
\end{enumerate}
